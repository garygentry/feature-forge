**Feature Forge** is a plugin that turns a feature idea into shipped, verified
code through a fixed, resumable **pipeline**. Each stage is a slash command you
invoke; state persists between stages, so you can stop and resume anytime.

<Diagram name="arch" alt="The Feature Forge pipeline: setup, optional epic, then the six stages PRD, tech spec, implementation specs, backlog, autonomous loop, and docs, with a verify-and-fix gate available after each stage." />

## The pipeline at a glance

You write a **PRD** (what to build); Feature Forge turns it into a **tech spec**
(how), then granular **implementation specs**, then a **backlog** of self-contained
work items. An **autonomous loop** implements the backlog item by item — running
your tests and committing as it goes — and finally generates **docs**. A
**verify & fix** gate is available after every stage.
